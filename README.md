# ModUA

Modbus to OPC UA bridge / utility.

簡介
- 這個專案將 Modbus 裝置與 OPC UA 整合，可作為橋接或工具使用。

快速開始
1. 建議建立虛擬環境並安裝套件：
```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```
2. 執行應用程式：
```powershell
python app.py
```

注意
- 本倉庫已忽略常見的暫存檔與安裝程式（見 `.gitignore`）。
