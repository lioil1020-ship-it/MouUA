## 核心模塊職責

### 1️⃣ **UI 層** (`app.py` + `ui/`)

#### `app.py` - 主應用程式
| 職責 | 方法/屬性 | 說明 |

| **窗口初始化** | `__init__()` | 初始化主窗口、tree、table、menu bar |

| **項目操作** | `new_project()`, `open_project()`, `save_project()` | 項目生命週期管理 |
| **運行時控制** | `toggle_runtime()`, `start_polling()` | 啟動/停止 Modbus 輪詢 |
| **樹結構操作** | `on_new_channel()`, `on_new_device()`, `on_new_tag()` | 增加項目 |
| **數據展示** | `update_right_table()` | 根據選中節點更新表格 |

| **數據寫入** | `_execute_tag_write()` | 寫入標籤值到設備 |

                    │                             │

| 模塊 | 職責 | 關鍵類/函數 |
|------|------|------------|
| `modbus_monitor.py` | 監視器、生命週期管理 | `RuntimeMonitor`, `RuntimeMonitorSignals` |
| `modbus_client.py` | Modbus 客戶端 (async) | `ModbusClient` (wrapper for pymodbus) |
| `modbus_worker.py` | 輪詢工作線程 | `ModbusWorker`, `ModbusScheduler` |

#### 運行流程

```
用戶點擊 [🔴 Runtime]
        │
        ▼
IoTApp.toggle_runtime()
        │
        ▼
RuntimeMonitor.start()
        │
        ├─ _extract_all_tags() 
        │   → 遍歷 tree，收集所有 Tag
        │
        ├─ _group_tags_by_config()
        │   → 按 Channel driver + Device ID 分組
        │
        ├─ for each group:
        │   ├─ _create_modbus_client()  → ModbusClient 連接
        │   ├─ _create_worker_for_group() → ModbusWorker 啟動
        │   └─ ModbusWorker._poll_loop()
        │       └─ 循環讀取 → 解析 → 緩衝 → 發射信號
        │
        └─ tag_updated 信號 → UI 更新表格
```

## 數據流與交互

### 📨 流程 1: 項目載入

```
用戶選擇 Project.json
        │
        ▼
app.py: open_project()
        │
        ▼
AppController.import_project_from_json(filepath)
        │
        ├─ 解析 JSON
        │
        ├─ 建立 Channel/Device/Group/Tag QTreeWidgetItem
        │
        ├─ 儲存配置到 item roles (col 1-7)
        │   - col 1: Description
        │   - col 2: Device ID / Driver
        │   - col 3: Timing / Communication
        │   - ...
        │
        └─ UI 刷新樹狀結構
```

### 📊 流程 2: Modbus 輪詢

```
RuntimeMonitor.start()
        │
        ├─ _extract_all_tags()
        │   └─ tree walk → 收集 (tag_item, device_item, channel_item)
        │
        ├─ _group_tags_by_config()
        │   └─ 按 driver + device_id 分組
        │
        ├─ 為每組建立 Worker
        │   ├─ ModbusClient.connect(host, port)
        │   └─ ModbusWorker.start()
        │
        └─ 輪詢循環 (ModbusWorker._poll_loop)
           ├─ 按 scan_rate 調度讀取
           ├─ ModbusClient.read_holding_registers(addr, count)
           ├─ 解析結果 + 套用 scaling
           ├─ 儲存到 DataBuffer
           └─ 發射 tag_updated 信號
                   │
                   ▼
            IoTApp._on_tag_polled()
                   │
                   ▼
            UI 表格實時更新
```

### ✍️ 流程 3: 標籤寫入

```
用戶在表格雙擊/右鍵選擇寫入
        │
        ▼
WriteValueDialog 彈出
        │
        ▼
用戶輸入值 + 選擇 Function Code
        │
        ▼
IoTApp._execute_tag_write()
        │
        ├─ 查找 tag 項目
        │
        ├─ 提取 address, data_type, scaling
        │
        ├─ ModbusClient.write_register(func_code, addr, value)
        │
        └─ 回覆成功/失敗
```

### 📡 流程 4: OPC UA 標籤映射

```
用戶設定 OPC UA 參數
        │
        ▼
app.py: apply_opcua_settings()
        │
        ├─ 驗證 OPC UA 配置
        │
        ├─ 啟動 OPCServer 實例
        │
        ├─ 遍歷所有 Modbus tag
        │   └─ OPCServer.add_tag(tag_meta)
        │       └─ 建立 OPC UA 節點
        │
        └─ tag_updated 信號會自動調用
           OPCServer.update_tag() ← 保持同步
```

#### 集成步驟

1. **實現真實 OPC UA Server**
   ```python
   # 建議: core/OPC_UA/opcua_server.py
   
   class ModUAOPCServer:
       def __init__(self, config):
           self.server = asyncua.Server()  # 使用 asyncua 或 opcua 庫
           
       async def start(self):
           await self.server.start()
           
       async def add_tag_to_opcua(self, tag_path, tag_meta):
           # 動態創建 OPC UA 節點
           
       def sync_with_databroker(self, databroker):
           # 定期同步 Modbus 值到 OPC UA
   ```

2. **在 RuntimeMonitor 中集成**
   ```python
   class RuntimeMonitor:
       def __init__(self, ..., opcua_server=None):
           self.opcua_server = opcua_server
           
       def _on_tag_polled(self, tag_path, value):
           self._signals.tag_updated.emit(...)
           # 同步到 OPC UA
           if self.opcua_server:
               self.opcua_server.update_tag(tag_path, value)
   ```

3. **在 AppController 中初始化**
   ```python
   class AppController:
       def __init__(self, app):
           self.app = app
           self.opcua_server = ModUAOPCServer(app.opcua_settings)
   ```

4. **在 IoTApp 中管理生命週期**
   ```python
   class IoTApp:
       async def start_runtime(self):
           if self.opcua_server:
               await self.opcua_server.start()
           self.runtime_monitor.start()
           
       def stop_runtime(self):
           if self.opcua_server:
               await self.opcua_server.stop()
           self.runtime_monitor.stop()
   ```
